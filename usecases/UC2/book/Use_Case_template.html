General Comments#

  • It could be that the exposure / hazard or risk data is too heavy. For those cases we could think of the Use Case for a subset and use the introduction file to present the results for the whole Use Case.

  • For large files, we will probably use zenodo for the storage of the Use Case files and download it through the Use Case.

  • For smaller files (csv, vulnerability curves, …), data can be stored directly on github.
